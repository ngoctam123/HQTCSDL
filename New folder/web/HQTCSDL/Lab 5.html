Lab 05: Tải và cài đặt Mongodb trên Docker

• Tải và cài đặt Mongodb trên docker.

• Tải image Mongodb

docker pull mongo

• Kiểm tra image có trong docker

docker image ls

• Chạy docker container

docker run --name mongo -p 27016:27017 -d mongo

• Để khởi động container đã được tạo

docker exec -it mongo bash

• Chú ý: lệnh trên để cổng 27016 để ko bị trùng với cổng mongo chính. Kiểm tra kết nối với 3tmongo

• Sau khi cài đặt hoàn tất sẽ có container SQLServer
